# Stock Price Predictor 📈

## 📌 About
This is a basic stock price prediction project using machine learning techniques. It predicts stock prices based on historical data.

## 📂 Files
- apple_stock_predictor.ipynb → main model implementation
- stock predictor.pdf → project documentation

## ⚙️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## ▶️ How to Run
1. Install required libraries:
   pip install pandas numpy matplotlib scikit-learn

2. Open the notebook:
   apple_stock_predictor.ipynb

3. Run all cells to see predictions

## 📊 Features
- Data preprocessing
- Visualization of stock data
- Prediction using ML model
